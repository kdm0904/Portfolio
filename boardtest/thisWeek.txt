해야 할 것 :

- 게시판 번호가 삭제 되었을 경우 그 다음에 작성하는 글은 이전 글의 다음 번호여야 한다
- 로그인을 한 사용자만 게시판 글쓰기 사용이 가능하게 만들어야 함
- 수정/삭제시 비밀번호를 입력해야 업데이트 되도록(Ajax활용)
- 파일 올리기 기능(1개씩~여러개)
- 부트스트랩으로 스타일 만들기
- 아이디 중복 확인 기능
- 게시판 리스트의 글을 클릭할 때 조회수가 자동으로 늘어나야 한다. 

- 로그인을 해서 본인의 글이 아닌 경우에는 수정 삭제 기능 사용 불가로 만들기
- 댓글 입력 수정 삭제 읽기
- 검색기능 추가
